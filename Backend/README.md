<p align="center">
A back-end service for <a href="> RealTimeVotes </a> App
</p>

## Run Locally

Clone the project

```bash
  git clone https://github.com/sachinakhil/Backend
```

Go to the project directory

```bash
  cd Backend
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn run dev
```
